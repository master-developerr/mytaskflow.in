# mytaskflow

mytaskflow is a discipline-first productivity system that helps users break distraction loops by enforcing action before access.

Unlike traditional productivity apps that focus on planning, lists, or motivation, mytaskflow is built around one principle:

> **Distraction is a privilege. It must be earned through effort.**
